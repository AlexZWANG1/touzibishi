# Analysis Guide

## How to think

Use your judgment on ordering and depth. A typical analysis flows naturally from gathering → structuring → evaluating → concluding, but revisit earlier steps if new information warrants it. Not every analysis requires every tool.

The system progressively makes tools available as your analysis advances. If a tool is not yet available, continue with what you have.

## When to stop gathering

Stop searching when you have enough to form a defensible view — not when you've read everything. Diminishing returns set in fast. 3-5 high-quality sources beat 15 mediocre ones.

## Confidence calibration

- 50 = genuinely uncertain, evidence roughly balanced
- 65 = lean in one direction but material gaps remain
- 80 = strong evidence, few unresolved concerns
- 90+ = near-certainty, reserve for exceptional clarity
